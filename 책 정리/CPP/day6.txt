- 매개변수와 값으로 전달하기
값에 의한 호출
매개변수 --> 함수 안에서만 사용가능, 밖으로 빼려면 처리가 필요
매개변수의 변수선언 결합은 허용하지 않는다     ex) void fuc(int a,b) (안된다)

- 배열을 매개변수로 사용하는 것의 의미
->  배열의 주소, 어떤배열, 원소가 몇개인지 함수에 전달한다.
	int func (int arr[], int n);
	int func (int arr[n]); 은 안된다.
값이 바뀌지 않는경우 const를 붙여 받아오자

- 배열의 범위를 사용하는 함수
포인터를 사용하여 배열의 값들에 접근한다.
const int * pt = arr;
	pt가 지시하는 값은 변경할 수 없지만 pt는 다른 변수를 지시 할 수 있다.
int * const pt = arr;
	pt를 사용하여 값을 변경 할 수 있다. pt는 다른 변수를 지시 할 수 없다.
const_cast --> const형 변경시 사용

- 함수와 이차원 배열
int sum(int (*ar2)[4]), int size)
int sum(int ar2[][4]), int size) 와 같은 식으로 이차원 배열을 가져온다.
